# Is "Surplus Land" Really Surplus?

## Project Description
This project examines whether there was "surplus land" on the Kazakh steppe around 1910. It is based on a large-scale Russian imperial survey and explores how land use was understood differently by colonial authorities and Kazakh communities. 

To address this question, the project presents selected survey data and a simplified analysis. It approaches the topic through a series of research questions that help explain how land was used, interpreted, and classified.
